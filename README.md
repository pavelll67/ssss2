Simple REST API

Простое backend-приложение на Node.js с использованием Express.
Предоставляет базовый REST API для работы с пользователями.

🚀 Функциональность
Получение списка пользователей
Получение пользователя по ID
Создание пользователя
Удаление пользователя
🛠️ Технологии
Node.js
Express
JavaScript (ES6+)
